# Lulu | Chic Girly Home

Static storefront recreated from the Stitch project `Chic Girly Home` (`13839523755981297067`).

## Files

- `index.html` - Main storefront experience.
- `styles.css` - Responsive styling based on Stitch's Lulu design system.
- `app.js` - Product rendering and interactive shopping bag drawer.
- `assets/images/` - Local images downloaded from the Stitch-generated HTML.
- `stitch-artifacts/screens/` - Original HTML files downloaded from Stitch for the requested screens.
- `stitch-artifacts/images/` - Original Stitch screen screenshots downloaded with `curl -L`.

## Run

Open `index.html` directly in a browser. No build step is required.
